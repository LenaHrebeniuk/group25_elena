Чек-лист описывает кратко, что нам нужно протестировать и результат

Структура Чек-Листа:

1. Заголовок (пример: Оформление заказа через корзину)
Описание проекта (пример: Чек-лист для проверки полного цикла оформления через корзину. Для тестирования нужно добавить товар в корзину и перейти к оформлению)

2. Подзаголовок
Название функционала, отдельного элемента продукта (пример: Поле поиска)

3. Краткое описание теста
По каждому полю составляем краткий список проверок (пример: Ввести пароль латиницей)

4. Результат проверки – выполненные пункты отмечаются статусами
